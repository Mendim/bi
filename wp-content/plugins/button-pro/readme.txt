== Description ==

Button is a wordpress plugin. Create social icons and social share button. It is easy to use. You can create social sets of buttons. We have 7 social sets of buttons and 10 sets other css3 2D and 3D buttons.

Using [WD_Button id=XX] shortcode, you can publish Button in any Page or Post in your WordPress sites.

= Requirements = 
* PHP 5.3
* Wordpress installed

== Installation ==

1. Upload the entire **button** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Add new button, Use **[GPV id=456]** shortcode to publish your Button into any page or post.
